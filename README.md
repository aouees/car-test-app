# car-test-app
A mobile application designed to digitize vehicle inspection workflows and generate professional PDF reports.
The system features dynamic inspection checklists, an interactive car chassis interface for visual damage tracking, and automated image attachment for thorough documentation. It secures report storage via Google Drive integration for instant sharing and includes a robust search module to retrieve past inspection data by customer name or vehicle ID.
